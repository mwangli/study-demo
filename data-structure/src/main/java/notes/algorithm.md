##### 基础数据结构

- 数组：线性结构，通过索引访问
- 链表：单向链表，双向链表，环形链表，通过指针访问下一个元素
- 队列：先进先出，在队尾添加元素，在队首取出元素
- 栈：后进先出，入栈出栈都在栈顶位置
- 树：前序遍历，中序遍历，后序遍历
- 图：深度优先搜索，广度优先搜索
